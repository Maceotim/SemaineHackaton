Notre travail constiste en une interface utilisateur destinées à des gens non formés sur les outils de machine learning. 
Pour l'utiliser il suffit de d'ouvir XXXX, de charger un CSV contenant les variables avec lesquels l'utilisateurs veux entrainer un modèle.
Il faut ensuite sélectionner les input et les output désirés et notre programme livre le modèle entrainé avec ses indicateurs de performances ( r² et RMSE )
Important: Nous utilisons les bibliothèques pandas, numpy et sklearn.
